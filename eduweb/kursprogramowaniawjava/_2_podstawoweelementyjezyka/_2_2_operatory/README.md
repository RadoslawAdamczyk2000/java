# Operatory matematyczne i logiczne

## Operatory matematyczne:

- **+** - dodawanie,
- **-** - odejmowanie,
- **\*** - mnożenie,
- **&#92;** - dzielenie,
- **%** - modulo,

### Skrócone operatory matematyczne

```java
int a = 17;
double b = 4.0;
```

| Wyrażenie skrócone | Rezultat |
| ------------------ | -------- |
| a+=b               | a = a+b  |
| a-=b               | a = a-b  |
| a\*=b              | a = a\*b |
| a/=b               | a = a/b  |
| a%=b               | a = a%b  |

## Operatory logiczne

- **==** - równa się
- **!=** - różny
- **>=** - większy równy
- **<=** - mniejszy równy
- **>,<** - większy, mniejszy
